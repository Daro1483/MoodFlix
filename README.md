# MoodFlix

Sistema de gestión multimedia desarrollado con ASP.NET y SQL Server.

## Descripción

MoodFlix es una aplicación desarrollada bajo una arquitectura en capas que permite la gestión de contenido multimedia, usuarios y funcionalidades administrativas.

El proyecto fue realizado con fines académicos y de práctica profesional, aplicando conceptos de análisis funcional, diseño de software, seguridad y persistencia de datos.

---

## Tecnologías utilizadas

- C#
- ASP.NET
- SQL Server
- ADO.NET
- Enterprise Architect

---

## Arquitectura

El sistema está organizado en capas:

- UI
- Services
- DAL
- BE
- BLL
- Seguridad

---

## Funcionalidades principales

- Inicio de sesión
- Gestión de usuarios
- Gestión de películas
- Multiidioma
- Seguridad con dígito verificador
- Control de permisos
